== Description ==

= Output feed from the media library. Generate a podcast for iTunes Store. =

* Feeds of the audios and videos has been distributed as podcasts.
* Feeds of the image adds thumbnail.
* It can be displayed to custom post each feed using playlists.

== Installation ==

1. Upload `medialibrary-feeder` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
